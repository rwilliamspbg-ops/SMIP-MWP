================================================================================
  ANALYZING RESULTS
================================================================================

View latest benchmark output:
  cat benchmarks/bench-*.txt

Analyze CPU profile (requires Linux):
  go tool pprof -top benchmarks/bench-*cpu.prof

Web UI for flame graphs:
  go tool pprof -http=:8081 benchmarks/bench-*cpu.prof


================================================================================
  PERFORMANCE METRICS TO MONITOR
================================================================================

BenchmarkName-8    15234578        78.9ns/op   allocs/op=0, B/op=0

Key Metrics:
  - ns/op: Lower is better (throughput)
  - B/op: Lower is better (memory allocations)
  - allocs/op: Zero preferred for hot paths
  - Regression threshold: >5% sustained increase in ns/op


================================================================================
  STRESS TEST CONFIGURATION
================================================================================

Load Levels:
  low    - 5 iterations, 8 concurrent (baseline)
  medium - 8 iterations, 16 concurrent (normal load)
  high   - 12 iterations, 32 concurrent (production stress)

Custom run example:
  LOAD_LEVEL=high DURATION=600 CONCURRENT=64 ./scripts/stress-test.sh
